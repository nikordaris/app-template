# @app/api / generated

GraphQL operation types and helpers generated by Amplify / `ampx` live here. The template ships an empty tree until you run backend codegen.

## Refresh locally

1. Deploy or run an Amplify sandbox for `packages/backend`.
2. Run `yarn workspace @app/backend generate:graphql` to populate this folder.
3. Commit the generated files when your team tracks them in git, or gitignore them if you prefer local-only codegen.
